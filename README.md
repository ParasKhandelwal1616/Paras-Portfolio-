Welcome to my professional portfolio! This is a high-performance, visually immersive web application built with **Next.js 15**, **React 19**, and **Three.js**. It features interactive 3D elements, glassmorphism UI, and smooth physics-based animations to showcase my full-stack engineering journey and projects.

---

## 🌟 Key Features

- 🎨 **Immersive 3D Experience**: Integrated **Three.js**, **React Three Fiber**, and **Spline** for interactive backgrounds and technical stack visualizations.
- ✨ **Glassmorphism UI**: A modern, sleek aesthetic with frosted glass effects and vibrant gradients.
- ⚡ **Physics-Based Interactions**: Technical stack section utilizes **Rapier** for real-time physics simulations.
- 📱 **Fully Responsive**: Optimized for all devices, from mobile to ultra-wide displays.
- 🚀 **Performance Optimized**: Built with Next.js App Router for fast loading and SEO friendliness.
- 🎭 **Smooth Animations**: Powered by **Framer Motion** for staggered reveals and interactive card effects.

## 🚀 Getting Started

### Prerequisites
- Node.js (Latest LTS)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/ParasKhandelwal1616/Paras-Portfolio-.git
   cd Paras-Portfolio-/portfolio
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run the development server:
   ```bash
   npm run dev
   ```

4. Open [http://localhost:3000](http://localhost:3000) to view it in your browser.
